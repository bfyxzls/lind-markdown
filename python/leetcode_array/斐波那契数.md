斐波那契数，通常用 F(n) 表示，形成的序列称为斐波那契数列。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。也就是：
```python
F(0) = 0,   F(1) = 1
F(N) = F(N - 1) + F(N - 2), 其中 N > 1.
```
#### 递归：效率最低
```python
def fib(N):
    if N == 0:
        return 0
    elif N == 1:
        return 1
    return fib(N-1) + fib(N-2)
```
#### 自底向上计算：
自底向上通过迭代计算，把结果保存在数组中
```python
def fib(N):
    res = [0, 1]
    for i in range(2, N+1):
        res.append(res[i-1] + res[i-2])
    return res[N]
```
